# 候鸟300 二零二六报名助手

一个面向参与者的 web-only 报名资料整理网站，用于把候鸟300 二零二六招募细则和本地资料收集模板整理成可操作的流程。

线上地址：[https://hn300.lovstudio.ai](https://hn300.lovstudio.ai)

## 功能

- 报名方向与作品类型选择
- 报批资料、宣传物料、专项作品材料清单
- 文件命名结构生成
- 提交前检查清单，本机浏览器保存状态
- 本地选择文件、校验格式和大小、生成最终资料包 zip
- 百度网盘链接格式检查与最终提交文本复制
- 官方模板下载入口，活动承诺书可跳转 Vadaski 签名板完成 PDF 签署

## 隐私说明

所有文件处理都在浏览器本地完成。本站不会接收、上传或保存参与者选择的身份证、承诺书、视频、图片等报名材料。

## 开发

```bash
pnpm install
pnpm dev
```

本地访问：`http://127.0.0.1:5173`

## 构建

```bash
pnpm typecheck
pnpm build
```

构建产物位于 `dist/`，可部署到 Vercel、Netlify、Cloudflare Pages 或任意静态托管服务。
